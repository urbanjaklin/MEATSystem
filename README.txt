=============== HOW TO COMPILE THIS CODE ===============

In order to properly compile this code, open your 
command prompt or preferred shell in the project's main
directory. This means within the folder/directory named
MEATSystem.
Once in this directory, use the following command
to compile the java code:

javac -cp ./src;jar/json-simple-1.1.1.jar MEAT.java

================ HOW TO RUN THIS CODE ==================

In order to run the program, use the following command
(provided you are in the same directory as when 
compiling):

java -classpath ./src;jar/json-simple-1.1.1.jar MEAT



Note: If you are on a Linux/UNIX system, replace the
semi-colons (;) above with colons (:).
